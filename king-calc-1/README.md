## Limits

### Formal Definition : Pg 18

$\lim\limits_{x \to a} f(x) = L $

$|f(x) - L| < \epsilon \text{ whenever } 0 < |x - z| < \delta$


### Solving Proofs

$\lim\limits_{x \to 4}(2x - 3) = 5$

$|(2x - 3) - 5| < \epsilon \text{ whenever } 0 < |x - 4| < \delta$

$|(2x - 3) - 5| < \epsilon $

Simplify:

* $|2x - 8| < \epsilon $
* $2|x - 4| < \epsilon $
* $|x - 4| < \frac{\epsilon}{2} $


$\delta = \frac{\epsilon}{2}$
